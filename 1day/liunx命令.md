# Liunx操作命令

- 进入目录

  ```
  cd 目录名称
  ```

- 进入多个目录

  ```
  cd /目录名/目录名/目录名
  ```

- 返回上一级目录

  ```
  cd ..
  ```

- 返回多级目录

  ```
  cd ../../../../
  ```

- 查看当前目录的内容

  ```
  ls 
  ```

- 查看当前目录下隐藏的内容

  ```
  ls -a
  ```

- 清屏

  ```
  clear
  ```

- 创建文件夹

  ```
  mkdir 文件夹名称
  ```

- 创建文件

  ```
  touch 文件名
  ```

- 删除文件或者文件夹

  ```
  rm -rf 文件名或者文件夹
  ```

- 复制文件

  ```
  cp 当前文件/当前文件夹 复制之后的文件/路径
  ```

- 查看文件内容

  ```
  cat 文件名
  ```

- 查看当前文件所在的目录

  ```
  pwd
  ```
